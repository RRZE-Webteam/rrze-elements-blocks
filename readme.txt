Advanced design elements for WordPress BlockEditor.

== Description ==

RRZE Elements Blocks ermöglicht die Nutzung der RRZE Elements-Suite im Gutenberg BlockEditor. Die folgenden Elemente / Blöcke werden unterstützt:

- Accordion + Inner Accordion
- Content Width Limiter
- Tabs
- Insertion
- Fluid Text Columns
- IconBox & Counter
- Custom News (Falls Elements aktiviert ist)
- Call to Action Box
- Notice
- Alert (Hinweisbox)

== Installation ==

1. Laden Sie die Plugin-Dateien in das Verzeichnis `/wp-content/plugins/rrze-elements-blocks` hoch oder installieren Sie das Plugin direkt über die WordPress Plugin-Seite.
2. Aktivieren Sie das Plugin über die 'Plugins' Seite in WordPress.
3. Nutzen Sie die Blöcke innerhalb des Gutenberg Editors nach Bedarf.

== Frequently Asked Questions ==

= Wie benutze ich die Blöcke? =

Nach der Aktivierung des Plugins stehen die Blöcke im Gutenberg Editor unter der Kategorie RRZE Elements zur Verfügung.
